# Default Port 3030 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Change WebDoc's default listener port from 3000 to 3030 everywhere users and tests observe the default.

**Architecture:** Keep the existing YAML-driven launcher and configuration schema. Change only the fallback value and its user-facing/test representations; explicit ports and the Playwright fixture port remain unchanged.

**Tech Stack:** TypeScript, YAML, Vitest, Next.js 16.

## Global Constraints

- The default listener is `127.0.0.1:3030`.
- Explicit YAML ports continue to override the default.
- `tests/fixtures/webdoc.config.yaml` remains on isolated E2E port `3210`.
- Explicit custom-port tests using `8080`, `4321`, and `9876` remain unchanged.
